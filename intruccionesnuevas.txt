## 2. Ajustar los Estilos CSS
Archivo a modificar: css/style.css

Acción: Vamos a modificar el layout del contenido para que el texto y la galería se muestren uno encima del otro.

Busca la regla .content-wrapper y reemplázala por este código. También, ajusta la media query para que el diseño sea consistente en móvil.

CSS

/* --- Estilos para el contenido dentro del acordeón --- */
.content-wrapper {
    display: grid;
    /* UNA SOLA COLUMNA PARA APILAR LOS ELEMENTOS */
    grid-template-columns: 1fr; 
    gap: 2.5rem;
    padding: 2rem 0;
    text-align: left;
}
No necesitas añadir más CSS. Con estos cambios, la sección del Día 3 mostrará primero la descripción de los grupos de trabajo y, justo debajo, la cuadrícula de la galería de fotos, manteniendo todo limpio y profesional.
