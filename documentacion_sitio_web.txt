# Documentación del Sitio Web G4IT

Este documento detalla el funcionamiento técnico, los estilos y los recursos utilizados en el sitio web del proyecto.

---

## 1. Cómo Funciona Todo (Arquitectura General)

El sitio web está construido con HTML, CSS y JavaScript puros, siguiendo prácticas modernas para asegurar que sea mantenible y escalable.

### 1.1. Estructura de Archivos

El proyecto se organiza de la siguiente manera:

- `/index.html`: La página de inicio.
- `/pages/*.html`: Contiene las sub-páginas (Servicios, Portafolio, Sobre Nosotros, Contacto).
- `/css/style.css`: Una única hoja de estilos que controla toda la apariencia del sitio.
- `/js/main.js`: Un único archivo JavaScript que maneja toda la interactividad.
- `/assets/`: Carpeta que contiene todos los recursos, dividida en:
    - `bitacora/`: Imágenes de la bitácora, organizadas por día.
    - `cv/`: Currículums del equipo.
    - `icons/`: Iconos en formato SVG.
    - `logo/`: El logo del sitio.
    - `perfiles/`: Fotos de perfil de los miembros del equipo.

### 1.2. Arquitectura CSS (El "cómo se ve")

El archivo `style.css` usa un enfoque organizado:

- **Variables CSS:** En la sección `:root`, se definen colores, fuentes y tamaños globales. Esto permite cambiar la paleta de colores de todo el sitio modificando solo unas pocas líneas.
- **Layout Moderno:** Se utiliza **Flexbox** para alinear elementos complejos como la barra de navegación y **CSS Grid** para crear las cuadrículas de contenido (servicios, portafolio, equipo), lo que permite que el diseño sea adaptable a diferentes pantallas.
- **Estética Futurista:**
    - **Glassmorphism (Efecto Cristal):** La barra de navegación usa un fondo semitransparente con un filtro `backdrop-filter: blur(10px)` para crear un efecto de cristal esmerilado sobre el contenido que pasa por detrás.
    - **Fondo Animado:** El fondo del sitio no es una imagen, sino un conjunto de gradientes y partículas animadas con la librería `tsParticles` para crear una atmósfera de "neblina digital" sin sacrificar rendimiento.

### 1.3. Lógica de JavaScript (El "cómo interactúa")

El archivo `main.js` añade la interactividad:

- **Animaciones al Hacer Scroll:** Usa `IntersectionObserver` para detectar cuándo un elemento entra en la pantalla y aplicarle una animación de aparición (`fade-in`). Es una técnica muy eficiente.
- **Menú Móvil (Hamburguesa):** El menú de navegación se transforma en un icono de hamburguesa en pantallas pequeñas, que despliega el menú al hacer clic.
- **Acordeón de Bitácora:** Permite expandir y contraer las secciones de cada día en la página "Sobre Nosotros".
- **Galería de Imágenes:** Se usa la librería `SimpleLightbox` para mostrar las galerías de imágenes de la bitácora en un formato profesional.

---

## 2. Estilo Usado para Cada Sección

### 2.1. Cabecera y Navegación (`.main-header`)

- **Estilo:** Glassmorphism.
- **Posición:** Fija (`position: fixed`) en la parte superior para que siempre esté visible.
- **Layout:** Flexbox para alinear el logo a la izquierda y el menú de navegación a la derecha.
- **Efecto Hover:** Los enlaces del menú tienen una línea animada que aparece debajo al pasar el ratón.

### 2.2. Sección de Equipo (`.team-section`)

- **Layout:** CSS Grid adaptable (`grid-template-columns: repeat(auto-fit, minmax(250px, 1fr))`), que ajusta automáticamente el número de columnas según el ancho de la pantalla.
- **Tarjetas de Perfil (`.profile-card`):**
    - Fondo semitransparente y un borde sutil.
    - Las fotos de perfil son circulares (`border-radius: 50%`) con un borde de color de acento.
    - Tienen una sombra y un ligero desplazamiento hacia arriba al pasar el ratón (`transform: translateY(-10px)`).

### 2.3. Bitácora y Acordeón (`.bitacora-section`)

- **Layout General:** Centrado y con un ancho máximo para facilitar la lectura.
- **Acordeón (`.accordion`):**
    - Cada ítem (`.accordion-item`) tiene un botón (`.accordion-header`) y un contenido (`.accordion-content`).
    - La animación de despliegue se logra animando la propiedad `max-height` del contenido de 0 a un valor automático.
    - El icono de la flecha gira 180 grados cuando el acordeón está activo.
- **Contenido del Acordeón (`.content-wrapper`):
    - Se divide en una cuadrícula de dos columnas: texto a la izquierda y galería de imágenes a la derecha.
    - En móviles, la cuadrícula se convierte en una sola columna, apilando el texto sobre la galería.
